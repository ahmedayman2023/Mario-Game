# Feynman Study League (دوري فاينمان للمذاكرة)

تطبيق متطور لإدارة وقت المذاكرة باستخدام **تقنية فاينمان (Feynman Technique)**، مصمم بواجهة حماسية مستوحاة من أجواء ملاعب كرة القدم الأوروبية لتحفيز المستخدمين على التعلم العميق.

## 🚀 المميزات

- **نظام تقنية فاينمان المتكامل:** تقسيم الجلسة إلى خطوات علمية مدروسة بدون توقف بينها لضمان أقصى درجات التركيز.
- **المذاكرة النشطة (Active Study):** جلسة مذاكرة عميقة مدتها 90 دقيقة مقسمة إلى مشاهدة أولية (10 دقائق) ومذاكرة مركزة (80 دقائق).
- **واجهة "دوري الأبطال":** تصميم رياضي حماسي مع أصوات ملعب وتنبيهات "صفارة الحكم".
- **شريط تقدم تفاعلي:** يوضح الخطوة الحالية ووصفها العلمي.
- **قائمة تكتيكات (Todo List):** لإدارة المهام الفرعية داخل الجلسة.
- **أجواء الملعب:** إمكانية فتح خلفية صوتية وبصرية للملعب في علامة تبويب منفصلة.

## 📋 خطوات الجلسة

1. **اختر المفهوم (5 دقائق):** تحديد الموضوع وتفريغ المعلومات الأولية.
2. **المذاكرة النشطة - المشاهدة الأولى (10 دقائق):** تدوين الكلمات المفتاحية بتركيز عالي.
3. **المشاهدة والتوقف والكتابة (24 دقيقة):** مشاهدة بتركيز مع التوقف كل 3 دقائق لتدوين الملاحظات.
4. **المذاكرة النشطة - المذاكرة العميقة (56 دقيقة):** الغوص في التفاصيل وفهم "لماذا وكيف".
5. **اشرحه لطفل (15 دقيقة):** تبسيط المعلومة واستخدام التشبيهات.
6. **حدد الفجوات (15 دقيقة):** العودة للمصادر لسد الثغرات.
7. **المراجعة والتبسيط النهائي (10 دقائق):** الصياغة النهائية بكلمات بسيطة.

## 🛠️ التقنيات المستخدمة

- **React 19**
- **Vite**
- **Tailwind CSS**
- **Motion (Framer Motion)**
- **Lucide React (Icons)**
