# Feynman Study League (دوري فاينمان للمذاكرة)

تطبيق متطور لإدارة وقت المذاكرة باستخدام **تقنية فاينمان (Feynman Technique)**، مصمم بواجهة حماسية مستوحاة من أجواء ملاعب كرة القدم الأوروبية لتحفيز المستخدمين على التعلم العميق.

## 🚀 المميزات

- **نظام تقنية فاينمان المتكامل:** تقسيم الجلسة إلى خطوات علمية مدروسة مع استراحات لمدة دقيقتين بينها لضمان تجديد النشاط.
- **المذاكرة النشطة (Active Study):** جلسة مذاكرة عميقة مقسمة إلى مراحل مشاهدة ومذاكرة مركزة لضمان الفهم الشامل.
- **واجهة "دوري الأبطال":** تصميم رياضي حماسي مع أصوات ملعب وتنبيهات "صفارة الحكم".
- **شريط تقدم تفاعلي:** يوضح الخطوة الحالية ووصفها العلمي.
- **قائمة تكتيكات (Todo List):** لإدارة المهام الفرعية داخل الجلسة.
- **أجواء الملعب:** إمكانية فتح خلفية صوتية وبصرية للملعب في علامة تبويب منفصلة.

## 📋 خطوات الجلسة

1. **اختر المفهوم (1 دقيقة):** تحديد الموضوع وتفريغ المعلومات الأولية. (إنجاز: 5%)
2. **مشاهدة بدون توقف بدون كتابة فقط فهم (5 دقائق):** استيعاب الفكرة العامة للموضوع بتركيز كامل. (إنجاز: 20%)
3. **المذاكرة النشطة - المشاهدة الأولى (10 دقائق):** تدوين الكلمات المفتاحية بتركيز عالي. (إنجاز: 50%)
4. **المشاهدة والتوقف والكتابة (24 دقيقة):** مشاهدة بتركيز مع التوقف كل 3 دقائق لتدوين الملاحظات. (إنجاز: 70%)
5. **مراجعة لما سبق عن طريق الكتابة زي أيام زمان (20 دقيقة):** ترسيخ المعلومات عن طريق الكتابة اليدوية والتلخيص. (إنجاز: 90%)
6. **اشرحه لطفل (15 دقيقة):** تبسيط المعلومة واستخدام التشبيهات. (إنجاز: 100%)

## 🛠️ التقنيات المستخدمة

- **Vite**
- **Tailwind CSS**
- **Motion (Framer Motion)**
- **Lucide React (Icons)**
